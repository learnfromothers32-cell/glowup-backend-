## How to Add Screenshots

1. Take screenshots of each major feature
2. Save them to `screenshots/` directory
3. Use consistent resolution (1280x720 recommended)
4. Update the paths above to match your filenames
5. Keep file sizes under 500KB each (use compression)

### Recommended Tool
- [Percy](https://percy.io) for visual regression
- Browser DevTools for responsive screenshots
- Clean up browser UI before capturing
